# Automação de Cadastro de Produtos no Fakturama com Python (RPA – Robotic Process Automation)

Este projeto utiliza **Python,PyAutoGUI e Pandas** para automatizar o processo de cadastro de produtos no ERP **Fakturama**.  
A automação faz a leitura dos dados de uma planilha Excel (`Produtos.xlsx`) e preenche automaticamente os campos do sistema, ajudando a diminuir a interação humana.

## O que eu usei
- Python 3
- PyAutoGUI
- Pandas
- Fakturama ERP 

## Funcionalidades da Automação
- Abre o Fakturama de forma automática
- Leitura de produtos do Excel
- Preenchimento automático dos campos: ID, nome, categoria, preço, custo, estoque e imagem do produto
- Salvamento automático de cada produto

## Objetivo
Demonstrar como Python pode ser usado para criar automações simples e eficazes sem precisar de APIs e complexidade, apenas com controle de tela e automação de teclado/mouse.
